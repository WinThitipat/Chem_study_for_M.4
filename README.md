# Chemistry Study for M.4

เว็บไซต์สำหรับเรียนรู้ **วิชาเคมีระดับมัธยมศึกษาปีที่ 4** ตามแนวทางหลักสูตรการศึกษาไทย พัฒนาเพื่อช่วยให้นักเรียนสามารถทบทวนและทำความเข้าใจพื้นฐานของเคมีได้ง่ายผ่านเว็บเบราว์เซอร์

## ภาพรวมของโปรเจกต์

**Chemistry Study for M.4** เป็นเว็บไซต์การเรียนรู้ที่พัฒนาขึ้นเพื่อรวบรวมเนื้อหาพื้นฐานของวิชาเคมีระดับ ม.4 ตามหลักสูตรที่ใช้ในโรงเรียนของประเทศไทย

เว็บไซต์ถูกออกแบบให้มีโครงสร้างเนื้อหาที่อ่านง่าย เข้าใจได้รวดเร็ว และเหมาะสำหรับนักเรียนที่ต้องการทบทวนบทเรียน หรือเตรียมตัวสำหรับการสอบ

โปรเจกต์นี้ยังเป็นส่วนหนึ่งของการฝึกพัฒนา **เว็บไซต์ฝั่ง Frontend** และใช้เป็นผลงานใน **Portfolio ด้านเทคโนโลยีและการเขียนโปรแกรม**

## หัวข้อเนื้อหาที่มีในเว็บไซต์

เว็บไซต์ครอบคลุมหัวข้อสำคัญของ **เคมีระดับมัธยมศึกษาปีที่ 4**

### 1. โครงสร้างอะตอม

* อนุภาคย่อยของอะตอม (โปรตอน นิวตรอน อิเล็กตรอน)
* เลขอะตอม
* เลขมวล
* ไอโซโทป
* การจัดเรียงอิเล็กตรอน

### 2. ตารางธาตุ

* การจัดเรียงธาตุในตารางธาตุ
* หมู่ (Group)
* คาบ (Period)
* โลหะ อโลหะ และกึ่งโลหะ
* แนวโน้มสมบัติของธาตุ

### 3. พันธะเคมี

* พันธะไอออนิก
* พันธะโคเวเลนต์
* พันธะโลหะ
* โมเลกุลขั้วและไม่ขั้ว

### 4. โมลและสูตรเคมี

* ความหมายของโมล
* เลขอาโวกาโดร
* การแปลงระหว่างกรัม โมล และจำนวนอนุภาค
* สูตรเอมพิริคัลและสูตรโมเลกุล

## เทคโนโลยีที่ใช้

เว็บไซต์นี้พัฒนาด้วยเทคโนโลยีพื้นฐานของการพัฒนาเว็บ

* **HTML5** – โครงสร้างของเว็บไซต์

เว็บไซต์ถูกเผยแพร่ผ่าน **GitHub Pages** ทำให้สามารถเปิดใช้งานออนไลน์ได้โดยไม่ต้องมีเซิร์ฟเวอร์ Backend
