โมลและสูตรเคมี

Mole Concept · Avogadro's Number · Empirical & Molecular Formula

4.1 โมลคืออะไร?

🔢

โมล (Mole)

โมลคือหน่วยวัดปริมาณสาร (SI unit) ใช้สัญลักษณ์ mol

เปรียบเหมือน "โหล" แต่ใหญ่กว่ามาก — 1 โหล = 12 ชิ้น, 1 โมล = 6.022 × 10²³ อนุภาค!

เลขอาโวกาโดร (Avogadro's Number)
N_A = 6.022 × 10²³

อนุภาค/โมล (mol⁻¹)

เข้าใจขนาดของ N_A

ตัวเลข 6.022 × 10²³ ใหญ่มากแค่ไหน?

  • ถ้านับ 1 อนุภาค/วินาที ต้องใช้เวลา ~19 ล้านล้านปี
  • มากกว่าจำนวนทรายบนโลกหลายเท่า
  • มวลอิเล็กตรอน 1 โมล = เพียง 0.548 มิลลิกรัม

ทำไมต้องใช้โมล?

อะตอมและโมเลกุลมีขนาดเล็กมาก การนับเป็นอนุภาคเดียวๆ ไม่สะดวก โมลช่วยให้เราทำงานกับปริมาณที่วัดได้จริงเช่น กรัม และลิตร

4.2 การแปลงระหว่างกรัม โมล และจำนวนอนุภาค

แผนภาพการแปลงหน่วย

÷ M
⇄
× M
🧪
โมล (mol)

⇄
÷ N_A
⚛️
จำนวนอนุภาค (N)
สูตรสำคัญ
n = m / M   |   N = n × N_A

n = โมล (mol)  |  m = มวล (g)  |  M = มวลโมลาร์ (g/mol)  |  N = จำนวนอนุภาค

ตัวอย่างที่ 1 — มวล → โมล

น้ำ (H₂O) 36 กรัม มีกี่โมล?

① M(H₂O) = 2(1) + 16 = 18 g/mol
② n = m / M = 36 / 18 = 2 mol
ตัวอย่างที่ 2 — โมล → อนุภาค

2 mol ของ H₂O มีกี่โมเลกุล?

N = n × N_A = 2 × 6.022×10²³
= 1.204 × 10²⁴ โมเลกุล
ตัวอย่างที่ 3 — กรัม → อนุภาค

NaCl 58.5 กรัม มีกี่ไอออน Na⁺?

① M(NaCl) = 23 + 35.5 = 58.5 g/mol
② n = 58.5 / 58.5 = 1 mol
③ Na⁺ = 1 × 6.022×10²³ = 6.022 × 10²³ ไอออน

มวลโมลาร์ (M) ของสารทั่วไป

สารสูตรM (g/mol)

คาร์บอนไดออกไซด์CO₂44
เกลือแกงNaCl58.5
แคลเซียมคาร์บอเนตCaCO₃100

4.3 สูตรเอมพิริคัลและสูตรโมเลกุล

สูตรทั้ง 2 ประเภท

ชนิดสูตรความหมายตัวอย่าง
Empirical Formula
(สูตรเอมพิริคัล)		 อัตราส่วนอย่างง่ายที่สุดของอะตอม CH₂O
Molecular Formula
(สูตรโมเลกุล)		 จำนวนอะตอมที่แท้จริงในโมเลกุล C₆H₁₂O₆
ความสัมพันธ์
Molecular = (Empirical)ₙ × n

n = จำนวนเต็มบวก (1, 2, 3, ...)

ตัวอย่างความสัมพันธ์

สารEmpiricalMolecularn
กลูโคสCH₂OC₆H₁₂O₆6
น้ำตาลซูโครสCH₂OC₁₂H₂₂O₁₁12*
เอทานอลC₂H₆OC₂H₆O1
BenzeneCHC₆H₆6
ไฮโดรเจนเปอร์ออกไซด์HOH₂O₂2

* ซูโครสสูตรเอมพิริคัลไม่ใช่ CH₂O แท้จริง แต่ใช้เพื่อเปรียบเทียบ

ตัวอย่าง: หาสูตรเอมพิริคัล

สาร X มีองค์ประกอบโดยมวล: C = 40%, H = 6.67%, O = 53.33%

① สมมติมวลรวม 100 g → C = 40 g, H = 6.67 g, O = 53.33 g
② แปลงเป็นโมล: n(C) = 40/12 = 3.33, n(H) = 6.67/1 = 6.67, n(O) = 53.33/16 = 3.33
③ หารด้วยค่าน้อยสุด (3.33): C:H:O = 1:2:1
④ สูตรเอมพิริคัล = CH₂O (มวล = 30 g/mol)
⑤ ถ้ามวลโมเลกุลจริง = 180 g/mol → n = 180/30 = 6 → สูตรโมเลกุล = C₆H₁₂O₆

4.4 โมลกับปริมาตรแก๊สที่ STP

สภาวะมาตรฐาน (STP)

STP = Standard Temperature and Pressure
อุณหภูมิ 0°C (273.15 K), ความดัน 1 atm (101.325 kPa)

ที่ STP แก๊ส 1 โมลทุกชนิดมีปริมาตรเท่ากัน:

ปริมาตรโมลาร์แก๊สที่ STP
V_m = 22.4 L/mol

V = n × 22.4  |  n = V / 22.4

เหตุผล: ตาม กฎแก๊สสมบูรณ์ (Ideal Gas Law: PV = nRT) ที่ STP แก๊สทุกชนิดมีพฤติกรรมเดียวกัน

ตัวอย่างที่ 1

CO₂ 0.5 mol มีปริมาตรที่ STP เท่าไร?

V = n × 22.4 = 0.5 × 22.4 = 11.2 L
ตัวอย่างที่ 2

O₂ 3 mol มีปริมาตรที่ STP เท่าไร?

V = 3 × 22.4 = 67.2 L
ตัวอย่างที่ 3

แก๊สที่ STP มีปริมาตร 44.8 L มีกี่โมล?

n = V / 22.4 = 44.8 / 22.4 = 2 mol

สรุปการแปลงหน่วยทั้งหมด

จากไปสูตรตัวอย่าง

อนุภาคโมลn = N / 6.022×10²³1.204×10²⁴ → 2 mol

สรุปสำคัญ บทที่ 4 — โมลและสูตรเคมี
  • 1 mol = 6.022 × 10²³ อนุภาค (เลขอาโวกาโดร)
  • n = m/M  (n=โมล, m=มวล, M=มวลโมลาร์)
  • N = n × N_A  (จำนวนอนุภาค)
  • สูตรเอมพิริคัล = อัตราส่วนอย่างง่ายของอะตอม
  • สูตรโมเลกุล = (Empirical)ₙ × n
  • แก๊ส 1 mol ที่ STP มีปริมาตร = 22.4 L
